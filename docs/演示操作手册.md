## 6. 常见问题

### Reactscan 只有 DEPLOY 没有 callback

先检查：

- RC 是否是最新地址
- 订阅是否使用 `REACTIVE_IGNORE`
- Reactive / callback proxy reserves 是否充足

### B 链状态没有变化

先检查：

- `NearLiquidation` 是否真的发出
- `ProtectionExecutor` 地址是否与 RC 构造参数一致
- 查询的是否是最新 B 链执行器地址
